1. Curve Fitting by the Method of Least Squares

Analogy: The "Best Guess" Path

Imagine you're tracking a drone's flight path with several snapshots of its position. These snapshots are your data points. Due to wind and other factors, the points don't form a perfect line. Curve fitting is like drawing the smoothest, most probable path the drone is taking through these scattered points. The method of least squares is a formal way to find the single best possible path.

It works by minimizing the sum of the squared differences (residuals) between the observed data values and the values predicted by the curve. For a set of data points $(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)$ and a curve $y = f(x)$, we minimize:

Sum of Squared Residuals
$$S = \sum_{i=1}^{n} (y_i - f(x_i))^2$$

1.1 Fitting of Straight Lines (The Least-Squares Line)

Analogy: The Closest Rope

Imagine your data points are people standing in a field. You want to lay a long, straight rope on the ground so that it's as "close" as possible to everyone. The method of least squares finds the optimal placement by minimizing the sum of the squared vertical distances from each person to the rope. Squaring the distances ensures that errors don't cancel out and heavily penalizes points that are far away.

When data appears to follow a linear trend, we fit a straight line of the form:

Linear Equation
$$y = a + bx$$

The constants a (y-intercept) and b (slope) are found by solving the Normal Equations:

Normal Equations
$$\sum y = an + b\sum x$$ $$\sum xy = a\sum x + b\sum x^2$$

These can be solved using the explicit formulas:

Slope and Intercept Formulas
$$b = \frac{n\sum xy - \sum x \sum y}{n\sum x^2 - (\sum x)^2}$$ $$a = \bar{y} - b\bar{x}$$
Example: Study Time vs. Exam Score

A student tracks their study hours (x) and exam scores (y): (2, 65), (4, 75), (5, 85).

  1. Calculate Sums:
    • $n = 3$
    • $\sum x = 11$, $\sum y = 225$
    • $\sum x^2 = 45$, $\sum xy = 855$
  2. Calculate Slope (b): $$b = \frac{3(855) - (11)(225)}{3(45) - (11)^2} = \frac{90}{14} \approx 6.43$$
  3. Calculate Intercept (a): $$a = \bar{y} - b\bar{x} = \frac{225}{3} - 6.43\left(\frac{11}{3}\right) \approx 51.43$$
  4. The Least-Squares Line is: $$\text{Score} = 51.43 + 6.43 \times \text{Study Hours}$$ This equation can now be used to predict scores for different study times.

1.2 Fitting of Second-Degree Parabolas and More General Curves

Analogy: The Flexible Rope

If the people in the field are arranged in a curve, a straight rope is a poor fit. You now need a flexible rope that can bend into a parabola. The method of least squares finds the exact curve that best fits this pattern.

For a second-degree parabola, the equation is:

Parabolic Equation
$$y = a + bx + cx^2$$

The Normal Equations for a Parabola are:

Normal Equations for Parabola
$$\sum y = an + b\sum x + c\sum x^2$$ $$\sum xy = a\sum x + b\sum x^2 + c\sum x^3$$ $$\sum x^2y = a\sum x^2 + b\sum x^3 + c\sum x^4$$

2. Correlation and Regression

Analogy: Two Different Questions

Imagine observing two people, Alex and Ben, walking.

  • Correlation asks: "Are they connected? Do they tend to walk in the same direction? How strong is this tendency?" The answer is a single score, the correlation coefficient r.
  • Regression asks: "Can you give me a formula or a map? If I know where Alex is, can I predict where Ben will be?" The answer is a model‚Äîthe regression equation.

The Linear Correlation Coefficient (r) and Coefficient of Determination ($r^2$)

The linear correlation coefficient (r) measures the strength and direction of a linear relationship. The coefficient of determination ($r^2$) is the proportion of the variance in the dependent variable that is predictable from the independent variable.

Analogy: Explaining the Story

If the correlation (r) between study hours and exam scores is 0.9, then $r^2 = 0.81$. This means that 81% of the variation in exam scores can be explained by the variation in study hours. The other 19% is due to other factors (quality of sleep, exam difficulty, etc.).

Formula for r
$$r = \frac{n\sum xy - \sum x \sum y}{\sqrt{[n\sum x^2 - (\sum x)^2][n\sum y^2 - (\sum y)^2]}}$$

Rank Correlation (Spearman's Rho)

Analogy: Movie Critics

Two critics might give a movie different scores (e.g., 9/10 vs. 7/10), but if they both agree it's the #1 movie of the year, their rankings are strongly correlated. Spearman's rank correlation is perfect for this‚Äîit ignores raw values and focuses only on the order.

Spearman's Rank Correlation ($r_s$)
$$r_s = 1 - \frac{6 \sum d^2}{n(n^2 - 1)}$$

Where d is the difference between ranks for each pair, and n is the number of pairs.

3. Multiple Regression and Correlation

Analogy: A More Complete Prediction Map

Predicting an exam score with only study hours is useful but incomplete. Multiple regression creates a more powerful prediction map by including more variables, like attendance percentage and previous quiz scores. Instead of a 2D line, the model becomes a 3D plane or a higher-dimensional surface.

For two independent variables, x and z, the multiple linear regression equation is:

Multiple Linear Regression
$$y = a + b_1x + b_2z$$

The constants are found by solving the Normal Equations:

Normal Equations for Multiple Regression
$$\sum y = an + b_1\sum x + b_2\sum z$$ $$\sum xy = a\sum x + b_1\sum x^2 + b_2\sum xz$$ $$\sum zy = a\sum z + b_1\sum xz + b_2\sum z^2$$

4. Application in Analysis of Data

Analogy: A Data Scientist's Toolkit

Think of these methods as a detective's toolkit. The data points are clues.

  • Curve Fitting helps you see the underlying pattern.
  • Correlation tells you how strongly different clues are related.
  • Regression allows you to build a predictive model to forecast what might happen next.
Applications in Computer Science and Electronics
  • Machine Learning: Regression is a core algorithm for prediction tasks like forecasting stock prices, user behavior, or system load.
  • Signal Processing: Engineers fit curves to noisy signals to filter out interference and identify the true underlying signal.
  • Software Engineering: Regression can be used to predict the number of bugs in a software module based on its complexity (e.g., lines of code, number of functions).
  • Resource Management: In cloud computing, regression models predict future server demand based on past usage patterns, helping to allocate resources efficiently and reduce costs.
